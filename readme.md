# open3d-tester

Version: 0.8.0

Testing Open3D Python package from PyPI and Conda.

```shell
# Inside pip virtualenv
pip install open3d

# Inside conda virtualenv
conda install -c open3d-admin open3d

# Inside conda virtualenv
pip install open3d-python
```
